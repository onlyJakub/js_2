Pytania dotyczące JavaScript

1) Kiedy powstał JavaScript (podaj datę)?
JavaScript został stworzony 4 grudnia 1995 roku.
2) Dlaczego jest to język skryptowy?
JavaScript jest językiem skryptowym, ponieważ jest wykonywany bezpośrednio w środowisku, takim jak przeglądarka, bez konieczności kompilacji. Kod jest interpretowany dynamicznie podczas jego uruchamiania.
3) Gdzie jest wykonywany JavaScript (klient lub serwer)?
JavaScript pierwotnie był wykonywany po stronie klienta (przeglądarki), jednak dzięki technologiom jak Node.js, może być wykonywany również po stronie serwera.
4) Dlaczego JavaScript jest to bezpieczne narzędzie?
JavaScript działa w tzw. sandboxie, co oznacza, że ma ograniczony dostęp do zasobów systemu operacyjnego, dzięki czemu nie może bezpośrednio uszkodzić komputera użytkownika. Wprowadza też mechanizmy kontroli dostępu.
5) Czy jest to język obiektowy, jeśli tak, to dlaczego?
ak, JavaScript jest językiem obiektowym, ponieważ pozwala na definiowanie obiektów, które mogą przechowywać właściwości i metody.
6) Wymień zalety JavaScript:
Zaleta 1: Szybkość - Jest wykonywany bezpośrednio w przeglądarce bez potrzeby kompilacji.
Zaleta 2: Wszechstronność - Może być używany zarówno po stronie klienta, jak i serwera.
Zaleta 3: Interaktywność - Pozwala tworzyć dynamiczne interfejsy użytkownika.
7) Framework to:
Framework to gotowa struktura programistyczna, która pomaga w tworzeniu aplikacji poprzez udostępnienie zestawu bibliotek, narzędzi i wzorców ułatwiających programowanie.
8) Do aplikacji mobilnych można użyć Framework?
Tak, frameworki takie jak React Native, Flutter czy Ionic pozwalają na tworzenie aplikacji mobilnych przy użyciu JavaScriptu lub jego wariantów.
9) Aplikacje responsywne to:
Aplikacje, które automatycznie dostosowują swój wygląd i układ do różnych rozmiarów ekranów, takich jak komputery, tablety i smartfony.
10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu ‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"> Kod ten tworzy przycisk, który po kliknięciu uruchamia funkcję WinOpen(), która otworzy nową stronę internetową (adres URL podany w cudzysłowie).
11) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200"); Funkcja ‹code>window.close()‹/code> zamyka aktualnie otwarte okno przeglądarki, o ile zostało ono otwarte za pomocą JavaScript.

toolbar=no - ukrywa pasek narzędzi (toolbar) w nowym oknie, który normalnie zawiera przyciski nawigacyjne (np. przyciski wstecz, dalej, odśwież). directories=no – ukrywa przyciski katalogów w nowym oknie. W starszych wersjach przeglądarek można było zobaczyć pasek z katalogami, ale obecnie ten element jest przestarzały i często ignorowany. menubar=no – ukrywa pasek menu (menu bar) w nowym oknie, który normalnie zawiera opcje takie jak "Plik", "Edytuj", "Widok", itp. height=280 – ustawia wysokość nowego okna na 280 pikseli. width=160 – ustawia szerokość nowego okna na 160 pikseli. top=200 – ustawia pozycję okna 200 pikseli od góry ekranu. left=200 – ustawia pozycję okna 200 pikseli od lewej strony ekranu.
