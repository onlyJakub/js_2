Podstawy JavaScript

  1. Rodzaje okien dialogowych
    • Alert – wyświetla okno z komunikatem i przyciskiem "OK".
    • Prompt – okno umożliwiające użytkownikowi wpisanie danych, które zostaną zwrócone do programu.
    • Confirm – okno, które pozwala użytkownikowi potwierdzić lub anulować daną akcję.
  2. Konwersje liczb w JavaScript
    • parseFloat(zmienna) – konwertuje wartość zmiennej na typ zmiennoprzecinkowy (float).
    • parseInt(zmienna) – konwertuje wartość zmiennej na typ całkowity (integer).
  3. Zasady tworzenia nazw zmiennych
    1. Nazwa zmiennej musi zaczynać się od litery, znaku $ lub _, i może zawierać tylko litery, cyfry, $ oraz _.
    2. Wielkość liter ma znaczenie – zmienna i Zmienna to dwie różne zmienne!
    3. Unikaj używania słów kluczowych JavaScript jako nazw zmiennych, np. var, function.
  4. Typy zmiennych w JavaScript
    • Number – typ liczbowy, obejmuje zarówno liczby całkowite, jak i zmiennoprzecinkowe.
    • String – typ tekstowy, przechowuje ciągi znaków.
    • Boolean – typ logiczny, przyjmujący wartość true lub false.
    • Null – reprezentuje brak wartości.
    • Undefined – oznacza, że zmienna została zadeklarowana, ale nie przypisano jej wartości.
    • Object – bardziej złożony typ, który przechowuje kolekcje danych.
